TMR&A = Trash Material: Recognition & Analysis
A program that can map out and recognize trash materials.
Making it easier for authorities to find and collect such trash for recycling.

Goal: To use AI and Machine Learning to create a strong, independent program that will assist humans in locating trash.
Aim: To have a cleaner and greener world.

Future Vision:
-> Mapping System
-> Publicly Available Database
-> Multiple Platform (Android, iOS) Support
-> Drone integration
-> To implement newer and cleaner recycling methods.
